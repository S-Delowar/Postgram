## **ðŸš€ Quick Start!**

### **Prerequisites**
- **Node.js** (v18+)
- **Python** (v3.11+)
- **Docker & Docker Compose**
- **PostgreSQL** (or use Docker)

---

### **1. Clone the Repository**
```bash
git clone https://github.com/S-Delowar/Postgram.git
cd Postgram
```

### 2. Set Up Environment Variables
Create a `.env` file at the `root/backend` directory with the following environmental variables:
```
# Django
DJANGO_SECRET_KEY=your-secret-key
DEBUG=True
ALLOWED_HOSTS=*

# Database
DB_NAME=postgres
DB_USER=postgres
DB_PASSWORD=postgres
DB_HOST=db
DB_PORT=5432

# CORS
CORS_ALLOWED_ORIGINS=http://localhost:5173

# AWS S3 (Optional)
USE_S3=FALSE
AWS_ACCESS_KEY_ID=your-access-key
AWS_SECRET_ACCESS_KEY=your-secret-key
AWS_S3_BUCKET_NAME=your-bucket-name
AWS_REGION=your-region
```

### 3. Run the Backend
**Option-1**: with Virtual environment
```bash
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

**Option-2**: with Docker (Recommended)
```bash
cd backend
docker-compose up --build
```

### 4. Run the Frontend
```bash
cd frontend
npm install
npm run dev
```

- **Backend**: http://localhost:8000
- **Frontend**: http://localhost:5173
- **Admin Panel**: http://localhost:8000/admin
